Blender-Rig-01

2021年04月06日 501 字 大概 2 分钟

# 一、准备工作

插件：rigify

获取方法：

打开 Blender 首选项设置，在插件搜索栏，搜索 rig 找到 rigify

准备要绑定的模型，将顶点组等清空。

# 二、绑定

1、将模型调整到合适的比例 (推荐使用单位 M），Shift+A 创建一个 Human（Meta-RIg）。

// 也可以是其他 meta-rig 谷歌，后面还可以在骨骼属性中创建 rig 默认骨架。

2、将骨架和模型对齐。

小技巧：在骨架属性中的视图显示设置，可以让你更直观的观察骨架。如 In Front（显示在最前面）等。

// 要注意骨骼轴向和命名

3、创建 rig 控件

将骨架与模型对应后，选择骨架，在骨架属性点击创建 Rig 控制器。

// 创建后可以原本的 Meta rig 进行隐藏，如果遇到 bug 就可以利用重新生成 rig 控制器。

4、绑定

选择要绑定的模型，（可以单独绑，也可以一起绑定），最后选择控制器，使 rig 控制器处于活动物体。按下快捷键 Ctrl+P，选择生成父级附带自动权重。

进入姿态模式，拖到一下 rig 控制器。模型跟着动作，ok，绑定完成！

# 三、修改

1、如果 rig 控件位置不对，可以通过调整原本的 meta-rig 然后修改完重新生成即可。

2、如果已经绑定好了，然后只想改变一个骨骼，可以在骨架属性创建 rig 控制器下面找到高级选项选上覆盖。然后再点击创建 rig。

3、如果模型要修改权重，首先先选择 rig 骨架，然后按 shift 选择要刷权重的模型部件，进入权重模式。

笔试属性中，修改笔刷的属性来刷权重，add 是加，Subtract 是减，越红权重越大，越蓝权重越小。以此类推。

刷权重的时候姿势不是对称的时候，可以使用顶点组对称。
